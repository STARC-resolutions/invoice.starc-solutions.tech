 🏗️ Engineering Highlights

To meet the standards of the **S.T.A.R.C. Protocol**, several custom modules were developed to solve industry-standard glitches:

* **Crore-Level Geometry:** A re-engineered coordinate system (`const cw`) designed to prevent character overlapping in high-value, multi-crore transactions.
* **Focus Stabilization:** Implemented targeted DOM updates via `updateRowMath` to prevent re-rendering glitches, allowing for continuous, uninterrupted typing of product details.
* **Indian Locale Logic:** Strict formatting for the Lakh/Crore system and automated multi-tier GST (CGST, SGST, IGST) calculations.
